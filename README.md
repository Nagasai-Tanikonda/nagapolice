# Research Portifolio Naga

This repository contains a minimal React scaffold created with Vite.

Quick start

1. Install dependencies

   npm install

2. Start dev server

   npm run dev

Open the shown localhost URL in your browser.

Notes / next steps

- Consider switching to TypeScript: add tsconfig and rename files to .tsx.
- Add ESLint / Prettier and a CI pipeline.
- Add routes, components and state management as needed.
